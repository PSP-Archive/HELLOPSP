Hello World, PSP!(for GCC 4.0.0)
Source code

Runs only on PSP 1.00.
Place EBOOT.PBP at directory PSP\GAME\HELLOPSP\ of Memory Stick, and 
you will find HELLO WORLD at GAME -> MEMORY STICK of XMB.
To shutdown, push home key and in submenu select to exit.
All scratch code, compiled by ps2dev toolchain.


How to build:

mk.bat, outpatch can be used on win32/cygwin/ps2dev-toolchain.  If your environment 
is something different, rewrite/recompile them.  Before you use, please understand 
what they are, what they do, and confirm they are compatible with your envs. 

startup.s     startup code
hellopsp.c    main routine
bitmap.c      bitmap data
font.c        font data
pg.c          primitive graphics
pg.h          its header

mk.bat        win batch file for building

outpatch.exe  a tool patching elf file generated by gcc/ld, to make it psp elf
outpatch.cpp  its source

Run mk.bat and you get outp as elf file for psp. 
Place this outp onto EBOOT.PBP, and you will be able to run on PSP1.00. 
Please refer to related sites how to make EBOOT.PBP and how to run. 

Everywhere scamped work.  Beware!

Hello World, PSP!(for GCC 4.0.0)
ソースコード

PSP 1.00のみで動きます。
メモリースティックにディレクトリPSP\GAME\HELLOPSP\を作成し、その中に
EBOOT.PBPをいれて下さい。XMBのゲーム→メモリースティックから起動できます。
終了させるには、[HOME]キーを押して、終了して下さい。
スクラッチで書いたコードです。ps2devの開発環境を使用しました。

ビルド方法

mk.bat, outpatch は win32/cygwin/ps2dev-toolchain で使用することができます。
他の環境の場合には必要な部分を修正して下さい。使用する前に、各ファイルの内容や
動作などを詳細に把握し、自分の環境に合っていることを確認して下さい。

startup.s     スタートアップコード
hellopsp.c    メインルーチン
bitmap.c      ビットマップデータ
font.c        フォントデータ
pg.c          簡易グラフィクスルーチン
pg.h          そのヘッダ

mk.bat        ビルド用バッチファイル

outpatch.exe  gcc/ldから出力されたelfファイルをpsp用にパッチするツール
outpatch.cpp  そのソース

mk.batを実行することにより、psp用のelfファイルとしてoutpが出力されます。
これをEBOOT.PBPに組み込むことにより、PSP1.00で実行できます。EBOOT.PBPの
作成方法やPSPでの実行方法については、関連サイト等を参照して下さい。

至るところ手抜きなので注意。
